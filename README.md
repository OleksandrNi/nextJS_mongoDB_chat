## Getting Started

run the development server:

```bash
npm run dev
# or
yarn dev
```

Add to .env file

```bash
NEXTAUTH_SECRET='...'
MONGODB_URI=mongodb+srv://account:password@cluster***.mongodb.net/?retryWrites=true&w=majority
NEXTAUTH_URL='http://localhost:3000/'
```


Open [http://localhost:3000](http://localhost:3000).
